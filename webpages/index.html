What do you want to do?

  • Visit a web page for the first time
  • Request a temporary unlock of a page
  • Install the proxy's self-signed certificate as a CA certificate. This is required for you to use this proxy to visit websites over HTTPS.
  • Administrate the list of domains of each user. For admins only!
